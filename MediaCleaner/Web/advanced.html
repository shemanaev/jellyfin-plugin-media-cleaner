Advanced playback matching

Controls how played and not-played rules handle media watched before Jellyfin says it was added.

Counts playback even when Last Played is earlier than Date Added. This can help with upgraded or re-imported media where Jellyfin reset the added date. Risk: newly re-downloaded media can be deleted based on old watch history.

Library tag maintenance

Renames a tag on matching library items. Rules are not changed automatically.
